Orbshield v1.0 Kit

This is the OrbShield, a small circuit board which interfaces with the Arduino microcontroller to convert serial gaming devices to USB HID plug-and-play controllers. It was designed for the SpaceTec SpaceOrb 360, a 6-degree-of-freedom controller for 3d games, and currently supports the SpaceOrb 360 and the Spaceball 2000, 3000, and 4000 series of 3-d CAD controllers.

It has been tested to work with Arduino software version 0016.

http://orbduino.sourceforge.net/_images/inventory.jpg

  1. OrbShield board
  2. DB9 Serial Connector socket
  3. 20-pin IC DIL socket
  4. 2-position DIP switch
  5. 2x 68 Ohm resistors
  6. 2x 3.6v 0.5w zener diodes

  9. 2x 8-pin male PCB headers
