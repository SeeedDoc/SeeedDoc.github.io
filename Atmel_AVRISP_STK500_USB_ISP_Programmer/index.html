Atmel AVRISP STK500 USB ISP Programmer

AVRISP2.0 STK500 programmer communicates with PC via USB port, and it works with most of PC & notebook computers.

Note: It is chinese version with PL2303(USB to Serial).

Model: TOL102C3B

Features

  • It can supports a wide range of ATMEL AVR microcontrollor.
  • It can works with AVR Studio or WINAVR(GCC)
  • It works with ATMEL AVR Studio 4.13, you can upgraded it's firmware,in upgrade firmware Completed,it can work at a higher version of the AVR Studio and to be able to support more device,
  • Compact Size - 75mm x 57mm x 27mm

Application Ideas

  • AVR programming
  • AVR Burn Bootloader

Usage


Example:Burn Bootloader to Arduino UNO

1.How to connect : See the picture above

2.Install AVRISP driver in the CD .

3.Download AVR STUDIO4.19 and install it.

Caution : The AVR STUDIO4.13 software in the CD can no support ATmega328P.

4.Burn Bootloader to Arduino UNO

-> Open AVR STUDIO4.19

-> Connect to STK500

->Select Platform

->Select Device "ATmega328P"

->Select Bootloader > Path:\arduino-1.0\hardware\arduino\bootloaders\optiboot\optiboot_atmega328.hex

->Program
