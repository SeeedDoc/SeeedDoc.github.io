DIY Magic Mirror Assembled - Plug and Play Version

Who's the fairest of them all? Make your own Magic Mirror and find out. The Magic Mirror plays animations based on input from various sensors. Featuring four characters, each character responds to the sensor inputs with its own personality.

Features

  • Breathalyzer
  • Weather Forecast
  • Stock Performance
  • Doorbell mode with IP camera video feed
  • Controls lights and appliances with X-10 control (requires optional X-10 CM17A dongle)
  • Streams your favoriate Picasa Web Album
  • Customizable with your own videos and sounds
  • LED control (Indicator LEDs for the Breathalyzer or a flickering fake fire effect)
  • Runs on the PC, Mac, and Linux
  • Quick Setup program for non-technical users
  • Advanced setup program for power users
  • Software source code available (Adobe AIR Action Script 3.0) upon request with purchased kit
