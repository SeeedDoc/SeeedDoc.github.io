Seeeduino Mega Protoshield Kit

This is the 1st Protoshield for Seeeduino Mega which brought all pins from Seeeduino, even Arduino to the top! With this protoshield, users will find a large prototyping area, you can even use Xbee with this board to make it a Xbee shield.

Model: INT108B1P

http://www.seeedstudio.com/depot/images/large/product/protomega_LRG.jpg


Features

  • Supports both Seeeduino MEGA and Arduino MEGA
  • All pins are brought to the top level
  • Supports Xbee/GPS bee
  • Rest button brought to the top level
